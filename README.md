## Install

Add the following to your `componser.json`

```json
{
  "require": {
    "floatingpoint/wp-classes": "^1.0"
  },
  "repositories": [
    {
      "type": "composer",
      "url": "https://packages.fpbeta.com/php/"
    }
  ]
}
```

Then run

```sh
composer update
```
